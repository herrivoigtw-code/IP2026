# Exercícios de Programação em Java

Este repositório contém exercícios e exemplos de programação desenvolvidos em Java durante as aulas da disciplina de programação do curso de Ciência da Computação na Universidade FURB.

Os códigos presentes têm como objetivo praticar conceitos fundamentais da programação, como lógica de programação, utilização de variáveis, entrada e saída de dados, operações matemáticas e estrutura básica de programas em Java.

Os exercícios aqui apresentados foram desenvolvidos como parte das atividades de aprendizado, buscando reforçar a compreensão dos conceitos trabalhados em aula e aprimorar habilidades de resolução de problemas computacionais.
